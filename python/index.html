Type Conversion: The process of converting one data type to another data type is called type conversion.

In Python, we can perform two types of type conversion. 1. Implicit Type Conversion - Here, Python automatically converts one data type to another in order to avoid data loss. > e.g. 1 * 1.23 = 1.23 as there is no data loss 2. Explicit Type Conversion - Here, the user can convert the data type of a variable to the required data type by using the in-built functions int(), float(), str(), etc. > e.g. "Hello " + str(123)
